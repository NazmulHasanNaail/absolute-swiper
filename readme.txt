== Installation ==

AUTOMATIC INSTALLATION
1. Search for Absolute Swiper through 'Plugins > Add New' interface.
2. Find the plugin box of Absolute Swiper and click on the 'Install Now' button.
3. Then activate the Absolute Swiper plugin.
4. Create a new slider: Click on the 'Absolute Swiper' menu, click on the Add New. Then you can select post type as slide.
5. After selecting post type Publish it and then copy the shortcode from the Shortcode box on right Side below Publish Box and paste it into a post or a page.

MANUAL INSTALLATION
1. Download Absolute Swiper
2. Upload Absolute Swiper through 'Plugins > Add New > Upload' interface or upload Absolute Swiper folder to the '/wp-content/plugins/' directory.
3. Activate the Absolute Swiper plugin through the 'Plugins' menu in WordPress.
4. Create a new slider: Click on the 'Absolute Swiper' menu, click on the Add New. Then you can select post type as slide.
5. After selecting post type Publish it and then copy the shortcode from the Shortcode box on right Side below Publish Box and paste it into a post or a page.
